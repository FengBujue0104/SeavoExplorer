主板项目文件浏览器（SeavoExplorer）使用说明
版本 0.4.1

适用范围
本程序是一个 Windows 桌面工具，用于浏览以 S/M 编号命名的主板、子卡项目文件夹。程序可扫描配置的项目根目录，快速定位项目，预览常见工程文档，并创建标准化项目/版本文件夹结构。

主要功能
1. 项目扫描与分类
   - 自动识别以 S 或 M 开头、后跟 3~4 位数字的项目文件夹。
   - 支持可选的“_注释”后缀，例如 S001_样机、M1234_说明。
   - 支持主板/子卡分区显示、项目搜索、按编号排序、递归扫描子文件夹。
   - 支持项目置顶、隐藏项目，以及恢复已隐藏项目。

2. 文件树浏览
   - 单击左侧项目后，在右侧文件树中显示项目内容。
   - 单击文件可预览内容并查看元数据。
   - 双击文件使用系统默认程序打开。
   - 双击文件夹或空白区域可在 Windows 资源管理器中打开对应目录。

3. 文件预览
   - 文本文件：.txt、.csv、.log、.bom、.drc、.rep、.rpt、.md、.json、.xml、.html、.htm、.ini、.cfg
   - PDF 文件：.pdf（支持多页预览）
   - Excel 文件：.xlsx、.xlsm、.xls
   - Word 文件：.docx、.doc
   - 图片文件：.jpg、.jpeg、.png、.bmp、.gif、.tiff、.tif、.webp、.svg
   - 视频文件：.mp4、.avi、.mov、.mkv、.flv、.wmv、.m4v、.webm、.mpg、.mpeg、.3gp（5 帧截图，在 10%/30%/50%/70%/90% 位置；需 OpenCV）
   - 压缩包文件：.zip、.rar、.7z（树状结构展示）
   - 加密/专用工程文件：.opj、.dsn、.sch、.brd、.dbk、.dsnlck 不直接解析，会显示不可预览提示，请使用对应 EDA 软件打开。
   - 预览开关：在“设置 → 预览设置”中可按类型（文本/PDF/图片/视频/压缩包/表格/文档）分别关闭自动预览；关闭后点击该类文件不立即读取，改为显示“显示预览”按钮，适合大文件或慢速磁盘。

4. 文件操作
   - 支持复制、粘贴副本、重命名、移入回收站。
   - 支持 Ctrl/Shift 多选后批量复制、批量压缩、批量移入回收站。
   - 粘贴副本会自动处理重名，避免覆盖原文件。
   - 删除操作会移入系统回收站，而不是直接永久删除。

5. 压缩包操作
   - .zip 使用 Python 内置能力进行预览和解压。
   - .rar 和 .7z 依赖外部 7-Zip（7z.exe）进行列表读取和解压。
   - 支持“智能解压”：包内只有一个顶层项目时直接解压到当前目录；多个顶层项目时自动创建同名文件夹，避免文件散落。
   - 支持将单个或多个选中项目添加到 zip 压缩包。

6. 检查更新
   - 点击“帮助 → 检查更新”即可从 GitHub Releases 检查最新版本。
   - 新版本弹窗会显示版本号、更新文件大小和发布页链接，可选择应用内下载或用浏览器打开发布页。
   - 应用内下载支持后台进度显示、重试和断点续传；网络较差时也可以直接从发布页手动下载。

7. 快捷访问栏
   - 可在“设置 → 快捷访问设置”中配置常用目录按钮。
   - 普通快捷按钮会在程序文件树中显示目录内容。
   - “不显示预览”按钮会直接用资源管理器打开，适合磁盘根目录、网络目录等大目录。

8. 新建项目与版本结构
   - “新建项目文件夹”：按 S/M 类型、编号、注释和目标位置创建项目根目录。
   - “新建文件夹内部结构”：在选中项目内创建版本目录（如 V01）和标准子文件夹（BOM、SCH、物料、评审、信号测试），也支持自定义子文件夹。
   - 上次选择的内部结构模板会自动保存，便于重复使用。

9. 视频预览与大图查看（0.4.0 新增/改进）
   - 多帧视频预览：视频文件显示 5 张截图，分别在 10%/30%/50%/70%/90% 时间位置。
   - 视频默认关闭：视频预览默认关闭，需在“设置 → 预览设置”中手动开启。
   - 视频帧查看器：点击预览图可打开帧查看器，左右方向键（← → ↑ ↓）或底部「◀ 上一张」「下一张 ▶」按钮切换帧。
   - 大图缩放与拖拽：图片/视频帧大图支持滚轮缩放、底部「−」「+」「1:1」按钮缩放、左键拖拽平移。
   - 面包屑路径栏：文件树上方显示从项目根到当前点选项的路径，点击中间任意一段可在文件树中快速定位到该文件夹；路径过长时中间自动省略。
   - 状态栏文件统计：选中项目后，状态栏右侧常驻显示该项目递归的文件数与总大小，在后台计算不卡界面。
   - 记住窗口与项目：退出时记住窗口大小/位置、左右分栏宽度、是否最大化，以及上次打开的项目，下次启动自动恢复；上次项目若已删除/改名/隐藏则安全跳过。
   - 已禁用全屏模式，避免菜单与关闭按钮不可见。

安装依赖
1. 确保已安装 Python 3.8 或更高版本。
2. 在项目目录中执行：
   py -m pip install -r requirements.txt

运行程序
1. 在命令行进入项目目录。
2. 执行：
   py main.py

打包程序
推荐使用仓库内置脚本：

1. 单文件 exe：
   py build_onefile.py
   输出文件：dist/SeavoExplorer.exe

2. 单目录分发包：
   py build_onedir.py
   输出目录：dist/SeavoExplorer/

3. PyInstaller spec 方式：
   pyinstaller main.spec
   输出文件名由 main.spec 中的 name 字段决定，目前为“主板项目文件浏览器”。

配置和数据文件
程序会在运行目录旁保存用户配置：
- seavoexplorer.json：项目路径、排序选项、快捷访问、7-Zip 路径、预览开关（text/pdf/image/video/archive/excel/word）、窗口大小/位置/分栏、上次打开的项目、隐藏/置顶项目等设置。
- seavo_comments.json：项目注释。

开发运行时，这些文件保存在脚本所在目录；打包为 exe 后，保存在 exe 所在目录。Windows 下程序会尽量将这些配置文件标记为隐藏，减少误删。

快捷键
- F5：刷新项目列表和文件树
- F2：重命名单个选中的文件/文件夹
- Ctrl+C：复制选中的文件/文件夹（支持多选）
- Ctrl+V：粘贴副本到选中文件夹或当前项目
- Delete：将选中的文件/文件夹移入回收站（支持多选）
- ← ↑：视频帧查看器中切换到上一帧
- → ↓：视频帧查看器中切换到下一帧

常见问题
1. 找不到项目
   - 检查项目根目录是否已在“设置 → 项目文件夹设置”中添加。
   - 检查文件夹命名是否符合 S/M + 3~4 位数字规则。
   - 如果项目位于更深层目录，请启用“包含子文件夹”。

2. .rar 或 .7z 无法预览/解压
   - 安装 7-Zip，或在“设置 → 7-Zip路径设置”中手动指定 7z.exe。
   - 程序会依次查找：设置路径、程序目录下的 7z.exe、系统默认安装目录。

3. 更新下载慢或失败
   - 先在“帮助 → 检查更新”里尝试应用内下载。
   - 如果网络较慢或 GitHub 下载不稳定，可直接打开发布页，用浏览器手动下载。
   - 断点续传会尽量利用已经下载的部分文件。

4. 大目录加载慢
   - 避免把磁盘根目录作为普通预览入口。
   - 可将大目录加入快捷访问，并勾选“不显示预览”。

5. 预览乱码
   - 文本预览会尝试 UTF-8 和 GBK 编码。
   - 若仍乱码，请用专业编辑器或文件对应软件打开。

6. 打包失败提示无法覆盖 exe
   - 通常是 dist/SeavoExplorer.exe 正在运行或被占用。
   - 关闭已运行的程序后重新执行打包脚本。
